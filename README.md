#Multimodal Classification 
A full-stack Django web application that leverages deep learning and machine learning to assist in the early detection, classification, and explainability of Lung Cancer, Pneumonia, Brain Tumors, and Dementia. Designed for research, educational, and clinical decision-support purposes with integrated AI transparency (Grad-CAM &amp; SHAP).

🖼️ Drag & Drop UI with real-time image preview
🔍 Auto-Grad-CAM: Dynamically detects the last Conv2D layer and generates attention heatmaps without manual layer configuration
📊 SHAP Gallery: Pre-computed interpretability plots (heatmap, beeswarm, waterfall, feature importance) for dementia risk factors
📡 JSON API Endpoints: Clean separation between frontend rendering and prediction inference
🛡️ Clinical Disclaimer: Professionally worded AI usage & validation warnings
📱 Fully Responsive: Optimized for desktop, tablet, and mobile clinical workflows

##Tech Stack
Backend: Python 3.10+, Django 4.2+
ML/DL: TensorFlow 2.x, Keras, scikit-learn, joblib, OpenCV, NumPy, PIL
Frontend: HTML5, CSS3, Vanilla JavaScript, Django Templates
Explainability: Custom GradientTape Grad-CAM implementation, SHAP pre-computed plots
Deployment Ready: WSGI/ASGI compatible, static file management, environment-based config

##How It Works
User Uploads Data: Image (PNG/JPG) for cancer/pneumonia/tumor, or clinical form for dementia.
Django Routes Request: views.py forwards payload to predictor.py.
Model Inference:
Images are preprocessed (224x224, normalized) and passed through loaded Keras models.
Tabular data is encoded & aligned to training column order before joblib inference.
Explainability Generation:
GradientTape computes class-specific gradients → pooled → resized → blended with original scan → base64 PNG.
Dementia returns SHAP plots & feature impact metadata.
JSON Response: {prediction, confidence, recommendation, gradcam: {success, gradcam_image}}
Frontend Rendering: JavaScript injects results + heatmap overlay with clinical disclaimers.

##Medical & AI Disclaimer
MedScan AI is strictly for research, educational, and decision-support purposes.
Predictions are generated by statistical models and do not constitute a medical diagnosis. AI outputs may contain false positives/negatives due to image quality, anatomical variance, or dataset bias. All results must be independently verified by a licensed radiologist, neurologist, or qualified healthcare professional before clinical action is taken.
