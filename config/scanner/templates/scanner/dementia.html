🧩 Dementia Risk Assessment

AI-driven cognitive decline prediction with SHAP-based explainability

1. SHAP Values Heatmap

Displays feature impact across all patients. Red indicates features pushing prediction toward dementia, blue indicates protective effects. EF, PS, age, and Global show the strongest consistent influence on model output.

2. Global Feature Importance

Ranking of features by mean absolute SHAP value. Ejection Fraction (EF) and Performance Status (PS) are the top predictors, followed by age and global cognitive scores.

3. Individual Patient Impact

Shows how each feature contributes to a specific patient's prediction. Bars pointing left decrease dementia risk, right increases it. Helps clinicians understand case-specific drivers.

4. Feature Value vs. Impact

Scatter plot showing how raw feature values correlate with SHAP impact. Higher age & lower EF/PS values cluster on the positive (risk) side, confirming clinical expectations.

5. SHAP Waterfall Plot

Step-by-step breakdown from baseline probability to final prediction. Each feature sequentially adjusts the output, providing a transparent audit trail for the AI's decision.
